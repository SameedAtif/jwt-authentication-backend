# JWT Authentication

## Overview

Basic authentication using JWT, with emphasis on best practices.

## Setup pre-req

* `ruby '2.7.2'`
* `rails '~> 6.1.4', '>= 6.1.4.1'`

## Installation

* `rails db:create db:migrate`
* `rails s`

## Known issues

### Could not open library 'sodium'

Possible fix: `brew install libsodium`
